## Upgrade / install

- **ISO**: flash `kcoreos-X.Y.Z-x86_64-linux.iso` (verify `SHA256SUMS`).
- **kctl Linux**: extract `kctl-X.Y.Z-linux-x86_64.tar.gz` and run `./kctl --help` (glibc Linux x86_64, built via this repo’s Nix flake).
- **kctl macOS Intel**: extract `kctl-X.Y.Z-macos-x86_64.tar.gz` and run `./kctl --help`.
- **kctl macOS Apple Silicon**: extract `kctl-X.Y.Z-macos-aarch64.tar.gz` and run `./kctl --help`.
